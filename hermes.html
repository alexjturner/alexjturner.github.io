Hermes

Hermes is our group's dedicated high performance computing system.
In Greek mythology, Hermes was the messenger of the gods. Hermes could also fly and was the patron of alchemists. As such, he seemed like a fitting namesake for a high performance computing system used to study atmospheric chemistry.

System Summary

  • Compute: 24 nodes, 768 CPU cores
  • GPUs: 2× H100 GPUs and 24× A2 GPUs
  • Memory: 192 GB RAM per compute node and 1 TB RAM per GPU node
  • Storage: 1 PB usable RAID-6 storage
  • Interconnect: 100 GB/s InfiniBand

Detailed Description

  • Compute nodes:
    • 24 compute nodes
    • Dual Intel Xeon Gold 6226R processors (16 cores each, 2.9 GHz)
    • 192 GB RAM per node
    • 1× NVIDIA A2 GPU (16 GB) per node
  • GPU partition:
    • 2× NVIDIA H100 GPUs (80 GB each)
    • Dual Intel Xeon Gold 6526Y (16 cores each, 2.8 GHz)
    • 1 TB RAM
    • 7 TB local cache disk
  • Storage and networking:
    • 1020 TB RAID-6 shared storage
    • Mellanox MSB7790-EB2F InfiniBand switch
    • 36 ports
    • 100 GB/s full bi-directional bandwidth per port
  • Software environment:
    • Debian GNU OS
    • Slurm workload manager
